## About

ANIA Player is a revolutionary frame-based avatar playback system that brings AI-generated avatars to life with intelligent lip-sync and idle animations. Built for streamers, content creators, and developers who want professional animated avatars without the overhead of traditional video systems.

## Quick Start

1. Download the appropriate version for your OS
2. Install/extract ANIA Player
3. Double-click any `.ania` file to launch (player auto-associates with .ania format)
4. Configure microphone input in settings
5. Customize appearance and behavior as needed

## ANIA Format

`.ania` files contain frame-based avatar animations created using **ANIA Creators**. All avatars:
- Are AI-generated with full image rights compliance
- Must be validated and approved by ANIA administrators
- Can be obtained from the [ANIA Models Market](https://(soon)) (free and premium)
- Can be created by anyone using ANIA Creators

## System Requirements

- **OS:** Windows 10/11, Linux (kernel 5.4+)
- **RAM:** 512MB minimum (varies by avatar complexity)
- **CPU:** Dual-core 2.0GHz or better
- **Audio:** Any system-recognized microphone/audio interface
